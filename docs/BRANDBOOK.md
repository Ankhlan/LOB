# CRE Brand Book
## Central Risk Exchange

---

## 1. LOGO SYSTEM

### Primary Mark
Pure typography: **CRE.mn**

The domain is the brand. No icons, no symbols.
Simplicity is elegance. Authenticity through transparency.

### Logo Variations
- **Full Logo**: "CRE.mn" wordmark (primary)
- **Icon**: "CRE" abbreviated (for favicons, app icons)
- **Light**: Dark text on white background
- **Dark**: Gold text on charcoal background

## 3. TYPOGRAPHY

### Primary Font
**Inter** - Clean, professional, excellent number rendering

| Style | Weight | Usage |
|-------|--------|-------|
| Light | 300 | Logo, headlines |
| Regular | 400 | Body text |
| Medium | 500 | Numbers, data |
| Semibold | 600 | Emphasis |

### Font Stack
`
font-family: 'Inter', -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif;
`

### Mongolian Cyrillic
Full support for Cyrillic (А-Я, а-я)
- Primary: Inter
- Fallback: Noto Sans

## 5. UI PATTERNS

### Surfaces
- Corner radius: 6px
- Shadow: 0 2px 8px rgba(0,0,0,0.3)
- Border: 1px solid #3e4451

### Buttons
- Primary: Gold background, dark text
- Secondary: Transparent, gold border
- Hover: 10% lighter

### Trading
- Buy: Green (#98c379)
- Sell: Red (#e06c75)
- Numbers: Monospace

### Charts
- Up candle: #98c379
- Down candle: #e06c75
- Grid: #3e4451
- Labels: #abb2bf

---

## 6. ICONOGRAPHY

### Style Guidelines
- Line weight: 1.5-2px
- Corners: Slightly rounded
- Size grid: 16px, 24px, 32px

### Icon Categories
- Charts and Trading
- Deposits and Balance
- Security and Authentication
- Settings and Configuration
- Mobile and Notifications

Use standard icon libraries (Lucide, Heroicons).
Maintain consistent stroke width across all icons.
