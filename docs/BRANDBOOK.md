# CRE Brand Book

## 1. LOGO SYSTEM

### Primary Mark
Mountain/chart motif representing:
- Price movement upward
- Stability and permanence
- Mongolia's landscape heritage

### Logo Variations
- **Full Logo**: Icon + "CRE" wordmark
- **Icon Only**: Mountain chart symbol (for favicons, app icons)
- **Text Only**: "CRE" logotype (for space-constrained uses)
